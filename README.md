#  *1. Cancer Prediction*

### Objective

Predict whether a tumor is **Malignant** or **Benign** using the Breast Cancer Wisconsin Dataset.

### Key Features

* Label Encoding
* Missing Value Handling
* Feature Scaling
* Logistic Regression
* Random Forest Classification
* Confusion Matrix
* ROC-AUC Analysis
* Feature Importance Visualization

### Evaluation Metrics

| Metric    |
| --------- |
| Accuracy  |
